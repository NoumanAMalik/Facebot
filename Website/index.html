Monty's Purpose

Named after its development origins, Monty (Python) connects Discord users from all four corners of the Earth.

Finding your next lab partner, roommate, or best friend has never been so easy.

Features / How It Works

    The Important Stuff:

  • Get started with !monty network monty asks you the important questions to get to know more about you and set up your profile.
  • After networking, !monty connect provides you with a list of your closest matches' Discord contact information.

Extras:

  • !monty python try this one out for yourself!
  • Need help? !monty help is here for you.
